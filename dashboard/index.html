Key Finding
The multi-signal approach achieves the highest F1 score by combining behavioral patterns (XGBoost), structural network position (PageRank + graph features), and relational propagation. No single signal alone captures all fraud patterns — behavioral analysis misses network-based fraud, structural analysis lacks transaction context, and propagation alone has high false positives without signal grounding.
